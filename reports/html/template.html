1.3. About this Test Report

1.3.1. Purpose and Scope

This Test Report is applicable to Sequans CLOE platform, and focuses on Teseo MCU software integration. Sequans LTE User Equipment (UE) firmware which is Cat-M on this platform has its own dedicated integration.

1.3.2. Software under test

Teseo Version: BUILD-AR-149-dirty master commit#324dd6127a5d67195a96d0ba8467b540888a99d6 (Mon, 30 Jul 2018 09:22:14 +0200)

Modem software (MFW) version is given later in this Test Report using dedicated AT command.

1.3.3. Who Should Read this Test Report

This document is intended for engineers using Sequans CLOE platform during the development and test phases.
